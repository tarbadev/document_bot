Q:What is an embedding?
A:A numerical vector that represents the meaning of text so similar texts can be compared.

Q:What is RAG?
A:Retrieval-Augmented Generation, a technique where external knowledge is retrieved and added to an LLM prompt.

Q:What is LangChain?
A:A framework for building applications with large language models, including tools for retrieval, chains, and agents.

Q:What is LlamaIndex?
A:A data framework that helps connect external data sources to LLMs.

Q:What is a vector database?
A:A database optimized for storing and searching embeddings, such as Pinecone, Weaviate, or FAISS.

Q:What is the OpenAI API used for?
A:It provides access to GPT models, embeddings, and other AI tools for building applications.

Q:What is Next.js?
A:A React-based framework for building modern web apps with server-side rendering and API routes.

Q:What is Terraform?
A:An Infrastructure-as-Code tool to provision and manage cloud resources.

Q:What is Docker?
A:A platform for packaging applications into containers so they run consistently across environments.

Q:What is observability in AI apps?
A:The ability to monitor metrics, traces, and logs to ensure reliability, performance, and debugging in production AI systems.
